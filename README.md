# Интернет-магазин / E-commerce frontend (React)

Фронтенд интернет-магазина с клиентской маршрутизацией, поиском товаров через query params и адаптивным интерфейсом. Проект разрабатывается как шаблон для реального магазина и дальнейшего переиспользования.

---

## 📌 О проекте

Проект создается как шаблон фронтенда для интернет-магазина с возможностью дальнейшей кастомизации под конкретный каталог и бренд.

Я отвечала за:

- архитектуру приложения  
- маршрутизацию (React Router)  
- реализацию поиска через URL  
- адаптивную верстку (desktop / mobile)  
- организацию проекта с использованием TypeScript  

Цель — создать масштабируемый и переиспользуемый frontend интернет-магазина с понятной логикой пользовательского сценария.

---

## 🛠️ Стек технологий

- React  
- TypeScript  
- React Router  
- Tailwind CSS  
- Vite  

---

## 🚀 Реализованная функциональность

- клиентская маршрутизация  
- поиск товаров через query params  
- отображение каталога товаров  
- базовая фильтрация по строке поиска  
- адаптивный интерфейс (mobile / desktop)  
- структура e-commerce интерфейса (header, страницы, footer)

---

## 🧠 Архитектурные особенности

- использование URL как источника состояния поиска  
- controlled inputs для форм  
- компонентный подход  
- постепенная типизация через TypeScript  

---

## 🚧 Статус проекта

Проект находится в активной разработке.

Разработка ведется как практическая реализация шаблона для интернет-магазина с последующей адаптацией под реальные задачи.

Планируется:
- расширение фильтрации и сортировки  
- улучшение UX  
- развитие логики корзины  
- доработка UI и адаптивности
